# assets/avatar-source

Avatar art sources go here at runtime: AI-generated portrait iterations,
layer-separated PSDs, Clip Studio files, and Cubism Editor project files.

**Guides:** what to generate →
[`docs/avatar-art-brief.md`](../../docs/avatar-art-brief.md); how to prove it
animates → [`docs/style-feasibility-rig.md`](../../docs/style-feasibility-rig.md).

**Never committed.** This directory's contents are gitignored — art sources
are large binaries with unclear redistribution status. Only this README and
`.gitkeep` are tracked.
